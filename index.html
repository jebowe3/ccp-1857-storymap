About This Project

SAMPLE TEXT: This map was created using CSV-2-Storymap, which takes a user input csv file and generates thematically coded markers for each entry based on the "chapter" attribute in the spreadsheet. These values are then fed to the dropdown menu allowing for user filtration of the scrolling sidebar content.

Choose from the options in the drop-down menu to filter results by the type of entry. Use the scrollbar to read through the major events of each activist's life and travels. As you scroll, the map will pan and zoom to the location connected with each event in the person's life story.
